Released [on Steam](https://store.steampowered.com/app/3476240/Berghotel_Heist/) in March 2026. It's a 2-player coop horror game developed in [Unreal Engine](https://www.unrealengine.com/).

I developed this game with my friends [David Zulic](https://www.davidzulic.com/) and [Manuel Eisl](https://www.manueleisl.com/) using David's studio name "Bindus Games".

It was a true indie release. We did all marketing and PR work ourselves and joined several indie areas with the help of the amazing [creAT]( https://www.creat.events/) club.

Here is an overview about the tools we used (slide from a talk I did at the [Vienna Unreal Meet-Up](https://communities.unrealengine.com/events/details/epic-games-vienna-presents-unreal-meetup-may-2025/)):

![img-bhh-techstack](/assets/img/posts/berghotelheist_techstack.png)


I worked on **so many** different aspects of the game that it's impossible to list them. However, some new learnings I had during this project were:
- How to use **Animation Retargeting** in Unreal
- How to use **Modelling Editor Mode** to quickly create and modify meshes
- How to setup + unlock Steam **Achievements via OnlineSubsystems** (OSSv1)
